## 摘要
药物协同预测（DSP）旨在识别不同细胞环境下针对不同靶点的有效药物组合。然而，新型化合物的不断涌现导致分子骨架和尺寸的变化，使得药物协同数据在拓扑结构上呈现分布外（O.O.D.）偏移。现有工作依赖于分布内（I.D.）假设，无法处理O.O.D.偏移。为解决此问题，我们首次通过图大语言模型研究分布外泛化药物协同预测。然而，O.O.D.泛化的DSP具有高度非平凡性，带来了若干挑战：i) 如何发现与细胞靶点相关的结构和无关的分子表示；ii) 如何找到精确计算分子表示的最优图神经架构；iii) 如何联合利用LLM中的分子结构和语义信息。为应对这些挑战，我们提出OOD-GraphLLM，一种新颖的图LLM框架，通过统一方式联合优化分子图表示和生物医学语义语言表示，能够在O.O.D.设置下准确预测药物协同。此外，我们微调了生物医学LLM——DrugSyn-LLM，并采用检索增强的生物医学指令调优策略，将对齐分子拓扑信息和分子语义信息与基于语言的推理相结合，用于O.O.D.泛化的DSP。源代码（https://github.com/EkkoXiao/Bio-GraphLLM）和发布的模型（https://mn.cs.tsinghua.edu.cn/bio-graphllm/）均已公开，用户可下载模型资源并通过Web界面交互式使用该系统。

## Abstract
Drug synergy prediction (DSP) aims to identify efficacious drug combinations under various cellular contexts with different targets. However, the continual emergence of novel compounds results in variations in molecular scaffolds and sizes, causing drug synergy data to exhibit out-of-distribution (O.O.D.) shifts with respect to topological structure. Existing works rely on in-distribution (I.D.) assumption, failing to handle the O.O.D. shifts. To solve this problem, we study out-of-distribution generalized drug synergy prediction through a graph large language model for the first time. Nevertheless, O.O.D. generalized DSP is highly non-trivial, posing several challenges: i) how to discover structurally relevant and irrelevant molecular representations with respect to cell targets; ii) how to find the optimal graph neural architectures that accurately calculate molecular representations; and iii) how to jointly leverage molecular structural and semantic information in LLMs. To address these challenges, we propose OOD-GraphLLM, a novel graphLLM framework which is able to accurately predict drug synergy under O.O.D. settings via jointly optimizing molecular graph representation and biomedical semantic language representations in a unified manner. Furthermore, we finetune DrugSyn-LLM, a biomedical LLM, and employ a retrieval-augmented biomedical instruction tuning strategy to align molecular topological information and molecular semantic information with language-based reasoning for O.O.D. generalized DSP. Both the source code (https://github.com/EkkoXiao/Bio-GraphLLM) and released model (https://mn.cs.tsinghua.edu.cn/bio-graphllm/) are publicly available, where users are allowed to download model resources and interactively use the system through a web interface.

### 论文提出的方法论

**核心思想**：通过联合优化分子图表示与生物医学语义语言表示，构建一个统一的图LLM框架（OOD-GraphLLM），能够自动适应新药物的结构变化并融合细胞上下文信息。

**关键技术细节**（四个组件）：

1. **目标自适应解缠分子图编码（Target-Adaptive Disentangled Molecular Graph Encoding）**  
   - 将分子图表示分解为目标相关与目标无关两部分。  
   - 通过跨注意力机制将目标相关表示与对应的靶点蛋白嵌入（来自ESM-2）交互，生成目标自适应表示。  
   - 引入去相关损失（\( \mathcal{L}_{\text{decorr}} \)）强制不同靶点自适应表示之间的统计独立性，减少冗余。

2. **成对注意力图架构搜索（Pairwise Attentive Graph Architecture Search）**  
   - 对每个药物对，使用双向多头注意力融合对方药物表示。  
   - 将候选消息传递算子（如GCN、GIN、GAT等）投影到连续潜在空间，并添加分离正则化（\( \mathcal{L}_{\text{sep}} \)）防止算子表示坍塌。  
   - 通过自适应路由机制，根据药物表示动态选择算子权重，组合成最优GNN架构以计算分子表示。

3. **多层级上下文化细胞特征对齐（Multi-Level Contextualized Cellular Feature Alignment）**  
   - **结构级**：将细胞系的基因表达特征投影后，与分子图的每个原子特征拼接，形成上下文感知的分子图。  
   - **语义级**：利用细胞系的文本描述和基因表达，通过BERT投影器生成与LLM输入空间对齐的连续表示。

4. **检索增强生物医学指令微调的DrugSyn-LLM**  
   - 两阶段训练：  
     - 阶段I：生物医学指令微调（\( \mathcal{L}_{\text{inst}} \)），从DrugBank/ChEMBL检索药物属性描述，作为监督信号引导LLM生成领域知识。  
     - 阶段II：任务特定训练（\( \mathcal{L}_{\text{task}} \)），使用链式思维（CoT）提示，先输出协同类别，再输出分数区间，最后精确回归数值。  
   - 整体损失：\( \mathcal{L} = \delta \mathcal{L}_{\text{inst}} + (1-\delta) \mathcal{L}_{\text{task}} + \alpha \mathcal{L}_{\text{decorr}} + \beta \mathcal{L}_{\text{sep}} \)，其中\(\delta\)为阶段指示符。

**算法流程**（文字说明）：  
1. 输入两个药物分子图及细胞系信息。  
2. 经解缠编码得到目标无关和多个目标自适应表示，拼接形成药物表示。  
3. 通过成对注意力融合两种药物信息，再经自适应路由选择最佳GNN算子，计算最终分子表示。  
4. 结构级与语义级细胞特征对齐后，与分子表示、SMILES文本一起输入DrugSyn-LLM。  
5. 先指令微调注入知识，再任务微调输出分类和回归结果。

### 论文的主要结论与发现

1. OOD-GraphLLM在所有O.O.D.设置、所有数据集、分类和回归任务上均显著优于现有SOTA方法，证明了联合图表示与LLM在O.O.D.泛化中的有效性。

4. 模型能够通过靶点共享和子结构模式实现跨分子泛化，验证了目标自适应解缠设计的合理性。
